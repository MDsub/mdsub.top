# 3. 英文字幕获取

> 在 RARBG 下载到的片源基本可能有三种：

1. 内置了字幕文件的 mkv （最常见）
2. 提供了字幕文件的 mp4 （少见）
3. 没有提供字幕文件的 mp4 或 mkv（极少见）

本篇将指导各位处理 1 和 3 两种情况
>
